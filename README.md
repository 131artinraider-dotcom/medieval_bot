# 🏰 شوالیه‌ی قرون وسطی (KnightLord RPBOT)

یک بات RPG قرون وسطایی کامل برای تلگرام

## ✨ ویژگی‌ها
- ۴ کلاس مختلف (شوالیه، سامورایی، اساسین، پالادین)
- سیستم نبرد ترن‌بیس با بونوس سلاح‌ها
- اینونتوری و تجهیزات
- فروشگاه با خرید/فروش
- دانجن‌های گابلین و ترول
- آپگرید استت‌ها
- دوئل ۵۰/۵۰
- ماموریت‌های روزانه
- لیدربرد گلوبال و گروهی
- جایزه کلمه کلیدی

## 📌 کامندها
- `/start` - شروع بازی
- `/status` - وضعیت شخصیت
- `/inventory` - اینونتوری
- `/shop` - فروشگاه
- `/dungeon` - دانجن
- `/upgrade` - آپگرید
- `/leaderboard` - رتبه‌بندی
- `/duel` - دوئل
- `/daily` - ماموریت‌ها
- `/help` - راهنما
